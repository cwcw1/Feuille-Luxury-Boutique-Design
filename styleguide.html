05. Motion Design

Animation System

Page Transitions

Smooth black overlay transitions between pages using GSAP.

gsap.to(transitionLayer, { y: "0%", duration: 0.6, ease: "power3.inOut" })

Reveal Animations

Elements fade in and slide up as they enter the viewport.

SCROLL REVEAL DEMO
gsap.to(element, { opacity: 1, y: 0, duration: 0.8, ease: "power3.out" }) /* Initial state */ .reveal { opacity: 0; transform: translateY(30px); }

Parallax Scrolling

Images move at different speeds than the scroll for depth effect.

gsap.to(image, { y: "20%", ease: "none", scrollTrigger: { trigger: ".hero", start: "top top", end: "bottom top", scrub: 1 // Smooth 1-second delay } })

Hover Effects

Subtle scale and opacity transitions on interactive elements.

/* Product cards */ .product-card:hover .product-img { transform: scale(1.05); transition: transform 0.6s ease; } /* Buttons */ .btn:hover { opacity: 1; transition: all 0.4s ease; }

Infinite Carousel

GSAP-powered seamless infinite horizontal scroll.

gsap.to(track, { x: -totalWidth, duration: 30, ease: "none", repeat: -1, modifiers: { x: gsap.utils.unitize(x => parseFloat(x) % totalWidth) } })

Easing Functions

power3.out
Used for: Reveal animations, element entrances
power3.inOut
Used for: Page transitions, modal animations
power2.out
Used for: Filter animations, quick interactions
none
Used for: Parallax effects (linear movement)
